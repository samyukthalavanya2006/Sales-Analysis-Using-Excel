This project is an interactive sales dashboard created in Microsoft Excel using the Vrinda Store dataset.
The dashboard helps analyze sales performance, customer behavior, order trends, gender contribution, state-wise sales, order channels, and age-group insights for the year 2022.

The main goal of this project is to transform raw sales data into meaningful business insights using Excel tools such as:

Pivot Tables
Pivot Charts
Slicers
Data Cleaning
Dashboard Designing
KPI Analysis
Dashboard Features

✅ Orders vs Sales Analysis
✅ Sales by Gender
✅ Order Status Distribution
✅ Top 5 States by Sales
✅ Orders: Age vs Gender
✅ Orders by Channels
✅ Interactive Slicers (Month, Channel, Category)

Business Questions Solved
Compare sales and orders using a single chart
Which month achieved the highest sales and orders?
Who purchased more: Men or Women?
What are the different order statuses?
Which states contribute the highest sales?
Relation between age and gender based on orders
Which sales channel contributes the most?
Highest selling category analysis
Key Insights
Women customers contributed around 64% of total sales.
Maharashtra, Karnataka, and Uttar Pradesh are the top-performing states.
Adult age group (30–49 years) contributes the highest number of orders.
Amazon, Flipkart, and Myntra are the major contributing channels.
Most orders were successfully delivered.
Final Conclusion

To improve sales, Vrinda Store should focus on:

Women customers
Age group 30–49 years
High-performing states like Maharashtra, Karnataka, and Uttar Pradesh
Marketing campaigns on Amazon, Flipkart, and Myntra
